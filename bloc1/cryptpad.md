## Instal·lació

Clonem el projecte.

```
sudo apt update
sudo apt install git
git clone https://gitlab.com/xtec/web/cryptpad
```

Accedim a la carpeta creada.

```
cd cryptpad
```

Executem el docker.

```
sudo snap install docker
docker compose up -d
```

Anem al navegador i obrim http://localhost:3000
