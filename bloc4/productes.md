# Woocommerce

Per a vendre productes mitjançant Wordpress un dels sistemes més utilitzats és el plugin Woocommerce. 

Aquest plugin permet gestionar els productes i vendes mitjançant la web de Wordpress.

Per a afegir el plugin anem a la secció de plugins->afegir plugin
