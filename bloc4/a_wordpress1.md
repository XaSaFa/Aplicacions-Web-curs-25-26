# Activitat: Creació d’un web comercial amb WordPress (en màquina virtual)

## Context

Disposeu d’una instal·lació nova de WordPress dins d’una màquina virtual. En aquesta activitat haureu de crear un web comercial d’una empresa fictícia (sector lliure: tecnologia, restauració, moda, producció audiovisual, gimnàs, acadèmia, etc.).

L’objectiu és practicar la gestió de WordPress: usuaris, entrades, pàgines, personalització i continguts legals.

No es lliurarà cap URL, ja que el web està en local. L’entrega serà un document amb captures de pantalla.

## Objectius d’aprenentatge

- Gestionar els diferents rols d’usuari.
- Diferenciar correctament entrades i pàgines.
- Configurar una portada estàtica.
- Personalitzar l’aparença d’un tema.
- Crear les pàgines legals obligatòries d’un web comercial.

## TASQUES OBLIGATÒRIES

# 1. Configuració inicial del lloc

**Tasques**

- Canvieu el títol del lloc pel nom de l’empresa.
- Afegiu una descripció curta (eslògan comercial).
- Configureu els enllaços permanents perquè siguin (simples).

**Pistes**
- Configuració del títol i descripció:
  - Tauler → Configuració → Generals
- Enllaços permanents:
  - Tauler → Configuració → Enllaços permanents → Nom de l’entrada

- Feu una captura de pantalla dels canvis.

# 2. Creació d’usuaris

- Heu de crear un usuari de cada tipus:
  - Administrador
  - Editor
  - Autor
  - Col·laborador
  - Subscriptor

**Condicions:**

- Els noms d’usuari han d’estar inspirats en actors o actrius de cinema (lliures).
- Cada usuari ha de tenir:
  - Nom visible
  - Correu electrònic amb format correcte
  - Rol assignat correctament

**- Pista (ruta del menú)**
- Tauler → Usuaris → Afegeix nou

Feu una captura de pantalla on es vegi la llista completa d’usuaris.

# 3. Creació d’entrades (Blog de l’empresa)

El web és comercial, per tant, totes les entrades han d’estar relacionades amb l’empresa.

**Heu de crear un mínim de 4 entrades.**

Exemples d’entrades possibles: 
- Presentació d’un nou producte o servei
- Notícia sobre l’empresa
- Cas d’èxit amb un client
- Promoció o oferta especial
- Esdeveniment organitzat per l’empresa
- Requisits obligatoris
- Cada entrada ha de tenir:
  - Títol
  - Mínim 200 paraules
  - Una imatge destacada
  - Categoria assignada
  - Almenys una etiqueta
  - Autor diferent en almenys dues entrades
**- Pistes (ruta del menú)**
- Crear entrada:
- Tauler → Entrades → Afegeix nova
- Categories i etiquetes:
- Menú lateral dret dins l’editor d’entrada
- Imatge destacada:
- Menú lateral dret → Imatge destacada

Feu captures de:

- Llista d’entrades
- Una entrada editada on es vegin categories i imatge destacada

# 4. Creació de pàgines obligatòries
## 4.1 Pàgina d’inici (portada estàtica)

Heu de crear una pàgina que serà la portada del web.

Ha d’incloure:
- Nom de l’empresa
- Text de presentació
- Mínim 2 imatges
- Enllaç a serveis o entrades
**- Pistes**
- Crear pàgina:
- Tauler → Pàgines → Afegeix nova
- Configurar com a portada estàtica:
- Tauler → Configuració → Lectura → Una pàgina estàtica

Feu captura on es vegi la configuració de la portada.

## 4.2 Política de privacitat

Ha d’incloure:

- Responsable del web (empresa fictícia)
- Dades que es recullen
- Finalitat del tractament
- Drets dels usuaris

**- Pista:**
- Tauler → Pàgines → Afegeix nova

## 4.3 Política de cookies

Ha d’explicar:

- Què són les cookies
- Tipus de cookies utilitzades
- Com desactivar-les

**- Pista:**
- Tauler → Pàgines → Afegeix nova

Feu captura de la llista de pàgines creades.

# 5. Personalització de l’aparença

Heu de modificar l’aspecte del web.

## 5.1 Canviar el tema

- Aneu a:
Tauler → Aparença → Temes
- Feu clic a Afegeix nou
- Trieu un tema adequat per a web comercial
- Instal·leu-lo
- Activeu-lo
- Feu captura del tema activat.

## 5.2 Personalitzar el tema

- Aneu a:
Tauler → Aparença → Personalitza
- Modifiqueu com a mínim:
  - Colors principals
  - Logotip (pugeu una imatge)
  - Capçalera o imatge destacada (si el tema ho permet)
  - Tipografia (si està disponible)
  - Publiqueu els canvis.
  - Feu captura del personalitzador.

## 5.3 Crear el menú principal

Heu de crear un menú amb:
- Inici
- Blog o Entrades
- Política de privacitat
- Política de cookies

**Pista:**
Tauler → Aparença → Menús

Assignar el menú a la ubicació principal del tema.

Feu captura del menú creat.

# 6. Ginys (widgets)

Afegiu un mínim de 2 ginys.

Exemples:
- Cercador
- Entrades recents
- Categories
- Text personalitzat

**Pista:**
Tauler → Aparença → Ginys

Feu captura on es vegin els ginys actius.

# ENTREGA

**Heu de lliurar un document (PDF o Word) que inclogui:**
- Captures obligatòries
- Configuració general
- Enllaços permanents
- Llista d’usuaris
- Llista d’entrades
- Una entrada editada
- Llista de pàgines
- Configuració de portada estàtica
- Tema activat
- Personalització
- Menú creat
- Ginys actius

Les captures han de ser clares, ordenades i amb títols explicatius.

**Criteris d’avaluació**

- Creació correcta dels rols
- Contingut coherent amb una empresa comercial
- Compleció de les pàgines legals
- Personalització visible del web
- Ús correcte de categories, etiquetes i imatges
- Ordre i qualitat del document d’entrega

Aquesta activitat simula la creació d’un web real d’empresa i us permetrà practicar les funcionalitats bàsiques de WordPress de manera aplicada i estructurada.
